## Install Ubuntu 24.04
- Open Windows Terminal app in Administrator mode
- `wsl -l -o`
- `wsl --install -d Ubuntu-24.04`
- Exit and open Terminal app in regular mode
- `wsl -d Ubuntu-24.04`
